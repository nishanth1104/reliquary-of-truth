# Reliquary of Truth — Architecture

## Core Principle
No work is considered complete unless it produces **verifiable proof**.

## High-Level Flow

User Task
↓
Intake Agent (clarifies & structures)
↓
Planner Agent (step-by-step plan)
↓
Owner Agent (code changes)
↓
Reviewer Agent (validates diffs)
↓
Verifier Agent (tests, logs, evidence)
↓
Delivered or Blocked

## Key Technologies
- Python
- LangGraph
- Pydantic
- Git
- Pytest
- OpenAI-compatible LLMs

## Why This Matters
Most AI systems generate answers.
Reliquary generates **work** — and proves it.
